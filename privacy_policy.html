개인정보처리방침

시행일: 2025년 6월 1일  |  최종 수정: 2025년 6월 1일

MIMIC Recorder(이하 "본 확장 프로그램")는 사용자의 개인정보를 소중히 여기며, 관련 법령에 따라 아래와 같이 처리합니다.

1. 수집하는 정보

본 확장 프로그램은 녹화 기능이 활성화된 동안에만 아래 정보를 수집합니다.

  • 사용자가 클릭한 웹페이지의 스크린샷(이미지)
  • 클릭이 발생한 페이지 URL
  • 클릭 좌표 및 타임스탬프

계정 정보, 이름, 이메일 등 개인 식별 정보는 수집하지 않습니다.

2. 정보의 이용 목적

  • 수집된 스크린샷을 AI(Claude)로 분석하여 액션 제목·설명을 자동 생성
  • 생성된 단계별 가이드를 사용자에게 제공

3. 정보의 저장 및 전송

  • 스크린샷은 Supabase 클라우드 스토리지에 업로드되어 저장됩니다.
  • AI 분석은 Anthropic Claude API를 통해 처리됩니다. API 키는 서버(Edge Function)에서만 관리되며 사용자 기기에 저장되지 않습니다.
  • 브라우저 로컬 스토리지(chrome.storage.local)에는 녹화 세션 동안 임시로 이미지가 저장되며, 전송 완료 후 자동 삭제됩니다.

4. 제3자 제공

수집된 정보는 아래 서비스 제공에 필요한 범위 내에서만 활용됩니다.

  • Supabase (supabase.com) — 이미지 저장 및 데이터베이스
  • Anthropic (anthropic.com) — AI 이미지 분석

마케팅, 광고, 기타 목적으로 제3자에게 정보를 판매하거나 제공하지 않습니다.

5. 정보의 보유 및 삭제

녹화 완료 후 로컬 캐시는 즉시 삭제됩니다. 서버에 저장된 이미지 및 메타데이터는 사용자가 요청 시 삭제할 수 있습니다. 문의: kinjungho@gmail.com

6. 사용자의 권리

  • 녹화는 사용자가 직접 시작·중지합니다. 비활성 상태에서는 어떠한 데이터도 수집하지 않습니다.
  • 수집된 데이터의 열람, 수정, 삭제를 요청할 권리가 있습니다.
